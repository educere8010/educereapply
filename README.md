# 에듀케레 입시상담앱 (내부용)

수시·정시 원서상담 도우미. **파일 하나로 끝나는 단일 HTML** 이며, 전국 전 계열 공식 입결 데이터가 앱 안에 내장되어 있습니다.

---

## 강사용 — 쓰는 법 (3단계)

1. 이 저장소에서 **`에듀케레_입시상담앱.html`** 을 내려받습니다.
   - 파일 이름을 클릭 → 오른쪽 **`Download raw file`**(아래 화살표 아이콘) 클릭.
2. 내려받은 파일을 **더블클릭** 합니다. (크롬/엣지 권장)
3. 끝입니다. 대학 데이터가 **자동으로 들어 있어** 바로 상담을 시작할 수 있습니다.

> 인터넷 연결이나 설치가 필요 없습니다. (아래 AI 기능만 인터넷·키 필요)

### 화면 순서
① **학생 입력** (내신·모의·정시 백분위·생기부) → ② **상담 결과** (수시 교과/학종/논술 배치 + 정시 배치 + 연도별 추세) → 인쇄(Ctrl+P)로 상담지 저장.

---

## AI 생기부 분석 (선택)

생기부 파일(PDF·이미지)을 AI가 읽어 강점·상담 코멘트를 써 주는 기능입니다.

- **④설정 탭**에 본인의 **Anthropic API 키**를 입력하세요. (console.anthropic.com 에서 발급)
- 키는 **각자 브라우저에만 저장**되며, 이 저장소·파일에는 들어가지 않습니다.
- 키 없이도 **최저 계산·배치·추세** 등 나머지 기능은 모두 정상 동작합니다.

---

## ⚠️ 보안 3원칙 (꼭 지켜주세요)

1. **API 키를 저장소/파일에 넣지 마세요.** 반드시 앱 ④설정에서 각자 입력합니다. (공개되면 유출·과금 위험)
2. **학생 생기부·개인정보 파일을 저장소에 올리지 마세요.** 학생 정보는 앱이 각자 브라우저에만 저장합니다.
3. 이 저장소는 **비공개(Private)** 로 유지하세요. 입결 데이터와 앱이 외부에 노출되지 않게 합니다.

---

## 관리자용 — 저장소 만들기 / 강사 초대

1. GitHub → **New repository** → 이름 입력 → **Private** 선택 → Create.
2. **Add file → Upload files** → `에듀케레_입시상담앱.html` 과 `README.md` 업로드 → Commit.
3. 강사 초대: **Settings → Collaborators → Add people** 에서 강사의 GitHub 계정 추가.
4. 강사는 초대 수락 후 위 "강사용 3단계"대로 파일을 내려받아 씁니다.

### 데이터 업데이트 (매년 요강 발표 후)
새 연도 입결·수능최저 자료가 나오면, 새 데이터가 내장된 `에듀케레_입시상담앱.html` 을 다시 만들어 **같은 파일을 덮어쓰기(Upload → Commit)** 하면 됩니다. 강사는 새 파일만 다시 내려받으면 됩니다.
